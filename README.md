Super Emulator

A modern and lightweight emulator project focused on performance, compatibility, and simplicity.

Features

- Fast and optimized emulation core
- User-friendly interface
- ROM loading support
- Audio and video emulation
- Save and load states
- Cross-platform support
- Open-source and customizable

Supported Systems

Current and planned systems:

- Nintendo Entertainment System (NES)
- Game Boy
- Game Boy Color
- Sega Master System
- More systems coming soon

Installation

Clone the Repository

git clone https://github.com/Brenninho123/Super-Emulator.git
cd Super-Emulator

Build

Follow the instructions for your platform and compiler.

Example:

mkdir build
cd build
cmake ..
cmake --build .

Usage

Launch the emulator and load a compatible ROM file.

SuperEmulator.exe

or

./SuperEmulator

Controls

Action| Key

Down| Down Arrow
